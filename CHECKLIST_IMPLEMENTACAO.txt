⚠️ TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ERRO: "Erro ao carregar os dados"
→ Verifique se dados.json existe e está no mesmo diretório
→ Valide JSON em jsonlint.com
→ Certifique-se que você está usando servidor web (não file://)

ERRO: Console mostra "CORS error"
→ Serve de um servidor adequado
→ Se usar cross-origin, configure headers CORS

PROBLEMA: Favoritos não salvam
→ Modo privado desativa localStorage
→ Tente em abeta normal
→ Verifique permissões do navegador

PROBLEMA: Layout quebrado em mobile
→ Limpe cache (Ctrl+Shift+Delete)
→ Teste em devtools responsivo (F12)
→ Atualize página (F5)
→ Teste em outro navegador

PROBLEMA: Animações não funcionam
→ Verifique se CSS carregou (F12 → Network)
→ Navegador suporta CSS3? (atualizar)
→ Desabilite extensões que bloqueiam CSS

PROBLEMA: Botões não respondem
→ Console mostra erros? (F12 → Console)
→ Script.js carregou? (Network)
→ JavaScript ativado?
→ Tente recarregar (Ctrl+F5)
